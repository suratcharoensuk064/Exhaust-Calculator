Exhaust Gas Flow Calculator

1. Fuel Mass Flow

ṁfuel = kg/s
\[ \dot m_{fuel} = \rho_{fuel} \times \dot V_{fuel} \]

2. Lambda

λ =
\[ \lambda = \frac{ CO_2 + \frac{CO}{2} + O_2 + \frac{NO}{2} + \left( \frac{H_{cv}}{4} \frac{3.5}{3.5 + CO/CO_2} - \frac{O_{cv}}{2} \right) (CO_2+CO) }{ \left(1+\frac{H_{cv}}{4}-\frac{O_{cv}}{2}\right) (CO_2+CO+nHC) } \]

3. Exhaust Density

ρ = kg/m³
\[ \rho = \frac{PM}{RT} \]

4. Exhaust Volume Flow (STD)

V̇STD = m³/s
\[ \dot V_{G,STD}=G' \times \dot m_{fuel} \]

5. Exhaust Volume Flow (Actual)

TSTD = 273.15 K
V̇actual = m³/s
\[ \dot V_{G,actual} = \dot V_{G,STD} \frac{T_{actual}}{T_{STD}} \]

6. Exhaust Mass Flow

ṁexhaust = kg/s
\[ \dot m_{exhaust} = \rho \times \dot V \]
